This website combines elements from multiple HTML5 UP templates, including Strata and Hyperspace. It has been customized to serve as my personal portfolio and showcase my work in physics and programming.

Features:
- Responsive design
- Configurable parallax background effect
- Poptrox-powered lightbox gallery
- Pre-styled elements
- Sass sources available

Demo images courtesy of Unsplash, a collection of CC0 (public domain) images.

Feedback, bug reports, and comments are welcome and appreciated.
